Ví dụ: Menu ngang cho bài tập 1

Đặt tệp này vào thư mục gốc của repo (thay thế index.html hiện có nếu cần). Menu đã được code theo dạng ngang, có trạng thái active cho trang hiện tại và responsive dùng nút hamburger trên thiết bị nhỏ.

Ghi chú nhanh:

  • Để active đúng trang: thêm class active vào thẻ <a> tương ứng (ví dụ: <a class="active" href="index.html">Trang chủ</a>).
  • Nếu bạn dùng nhiều trang (index.html, menu.html...), copy phần <header> vào từng trang để menu đồng nhất.
  • Nếu bạn dùng base URL khác trên GitHub Pages, kiểm tra đường dẫn href của menu cho phù hợp (ví dụ: prefix: /hce_demohtml1/ nếu cần).
